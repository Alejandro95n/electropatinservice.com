
ELECTRO PATIN SERVICE - INSTALACIÓN

1. Subir la carpeta 'theme-electropatin' a wp-content/themes/
2. Activar el tema desde el panel de WordPress.
3. Instalar y activar plugins desde la carpeta 'plugins'.
4. Importar productos con el archivo CSV en WooCommerce.
5. Subir imágenes en 'products-images' a la biblioteca de medios.
6. Configurar WooCommerce según las instrucciones.

Colores base: Rojo (#D32F2F), Blanco (#FFFFFF)
Plantilla basada en el diseño Taller ME.

Si tienes dudas, contáctame.
